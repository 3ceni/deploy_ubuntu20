1. подключаемся по smb

smb://10.6.6.119
вводим логин/пароль

2. из папки 
//10.6.6.119/Distrib/Linux/WP_Linux/cod
копируем в домашнюю папку скрипт "1start.sh"
(в нашем случае домашняя папка /home/rootmt, как только откроешь проводник по идее сразу попадаешь в домашнюю папку пользователя)

3. Отрываем терминал (CTRL+ALT+T) и последовательно вводим команды (попросит один раз ввести пароль для rootmt):

sudo chmod 644 1start.sh

sudo chmod +x 1start.sh

sudo ./1start.sh

4. Нужно задать пароль для AnyDesk и сохранить ID
